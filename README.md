# How to use the application:

This application is a student management system, so all of our functions will manipulate the student data.

### Add Student
to add a student, click the "Add Student" button on the left side of the interface. This will open up a window that will allow you to enter information about a student and will and them to the dataset.

### Remove Student
to remove a student, click the "Remove Student" button on the left side of the interface. This will open up a window that will allow you to search for a student via their ID and remove them from the dataset

### Edit Student
to edit a student's information, click the "Edit Student" button on the left side of the interface. This will open up a window that will allow you to enter a student's ID and change information regarding the student.

### Find Student
to search for a student, click the "Find Student" button on the left side of the interface. This will open up a window that will allow you to search for a specific student using their ID.

### Filter Students
to filter the data by majors or last names, interact with the search bars at the top of the interface. Once you begin typing the application will store your request and update the main display table.
